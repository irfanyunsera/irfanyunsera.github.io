Laboratory Activities

Lab 1 - UI Development

This lab introduced the fundamentals of building Android user interfaces using Jetpack Compose and modern declarative UI concepts.

I learned the basics of creating layouts using Jetpack Compose and how composable functions simplify UI development. Building reusable components helped me understand the structure of a modern Android application. This lab established the foundation for all subsequent mobile development tasks.

Lab 2 - Interaction & State Management

This activity focused on handling user interactions and managing application state effectively using Jetpack Compose.

Through this lab, I explored interactive components such as buttons and text fields while learning how state drives UI updates. Using remember and mutable state variables demonstrated the reactive nature of Jetpack Compose. It improved my understanding of creating dynamic user experiences.

Lab 3 - Material Design

This lab applied Material Design principles to create a more consistent and visually appealing Android interface.

I learned how Material Design components improve usability and consistency across applications. The use of themes, typography, and reusable design elements helped me create a more professional and polished interface. This lab emphasized the importance of user-centred design.

Lab 4 - Navigation

This activity introduced multi-screen navigation and architecture patterns for Android applications.

This lab taught me how to implement navigation between screens using NavHost and Navigation Compose. I also learned how ViewModel helps preserve and manage UI-related data across different screens. Understanding navigation architecture was essential for building scalable applications.

Lab 5 - Room Database Integration

This lab focused on local data persistence by integrating the Room database into an Android application.

I learned how to integrate Room Database to store and retrieve persistent data efficiently. The implementation of entities, DAOs, and repositories provided practical exposure to Android's recommended architecture. This experience strengthened my understanding of local database management.
